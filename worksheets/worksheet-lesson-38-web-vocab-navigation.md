# Lesson 38 — Web Vocabulary: Navigation
## विषय: इंटरनेट के शब्द — नेविगेशन

---

### Word List | शब्द सूची

| English | Hindi | What it means |
|---------|-------|---------------|
| **Home** | होम / मुख्य पृष्ठ | the main/starting page |
| **Back** | पीछे | go to previous page |
| **Next** | अगला | go to next page |
| **Search** | खोजो | find something |
| **Menu** | मेनू | list of options |
| **Login** | लॉगिन | enter your account |
| **Sign up** | रजिस्टर करो | create a new account |
| **Settings** | सेटिंग | change options |
| **Help** | मदद | get support |
| **Contact** | संपर्क | reach someone |

---
