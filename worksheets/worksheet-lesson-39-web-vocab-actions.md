# Lesson 39 — Web Vocabulary: Actions
## विषय: इंटरनेट के शब्द — काम के शब्द

---

### Word List | शब्द सूची

| English | Hindi | What it means |
|---------|-------|---------------|
| **Click** | क्लिक करो | press with mouse or finger |
| **Download** | डाउनलोड | save file to your device |
| **Upload** | अपलोड | send file to the internet |
| **Share** | शेयर | send to someone else |
| **Buy** | खरीदो | purchase something |
| **Add to cart** | कार्ट में डालो | select item to buy later |
| **Submit** | जमा करो | send your form or answer |
| **Delete** | हटाओ | remove something |
| **Save** | सेव करो | keep your work |
| **Print** | प्रिंट करो | print on paper |

---
