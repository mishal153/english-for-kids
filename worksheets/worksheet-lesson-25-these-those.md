# Lesson 25 — These / Those
## विषय: ये / वे

---

## Concept | अवधारणा

| Hindi | English | Use when |
|-------|---------|----------|
| यह (एक) | This | one thing, near |
| वह (एक) | That | one thing, far |
| ये (बहुत) | **These** | many things, near |
| वे (बहुत) | **Those** | many things, far |

---

### Exercise 1: Fill in the Blank | खाली जगह भरो

Use **this / that / these / those**. | सही शब्द लगाओ।

(पास में एक किताब) → ___________ is a book.

(दूर एक पेड़) → ___________ is a tree.

(पास में तीन कलम) → ___________ are pens.

(दूर दो कुत्ते) → ___________ are dogs.

(पास में एक बिल्ली) → ___________ is a cat.

(दूर पाँच बच्चे) → ___________ are children.

---

### Exercise 2: Match | मिलाओ

```
ये किताबें हैं     •          •   Those are dogs.
वे कुत्ते हैं      •          •   This is a mango.
यह आम है           •          •   These are books.
वह एक बिल्ली है    •          •   These are pens.
ये कलम हैं         •          •   That is a cat.
```
