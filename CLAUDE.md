# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Printable English workbooks for 13-year-old Hindi-speaking children who have no prior English knowledge. Goals: reading, writing, and speaking basics. Workbooks are Markdown files (.md) meant to be printed and completed with pencil on paper.

## Audience

- Age: ~13 years old
- Native language: Hindi
- English level: complete beginner
- Use Hindi translations/hints generously to make exercises accessible

## File Structure

One Markdown file per lesson or topic, e.g. `lesson-01-alphabet.md`, `lesson-02-greetings.md`.

## Exercise Formats

### Fill in the Blank
```
The color of the sky is __________.
आसमान का रंग __________ है।
```

### Hindi-to-English Matching
```
Draw a line to match:

नमस्ते    •        • Good morning
शुभ प्रभात •        • Hello
धन्यवाद   •        • Thank you
```

### Tracing (describe with underscores or dotted rows)
```
Trace the letter:

A  A  A  _  _  _  _  _
a  a  a  _  _  _  _  _
```

## Content Guidelines

- Keep sentences short and simple (max 5–6 words)
- Always provide the Hindi meaning below or beside English content
- Leave generous blank space for writing (use blank lines or underscores)
- Avoid idioms or culturally unfamiliar references

## Lesson Progression

Two series, each building on the previous. End goal: read and understand everyday website content.

### Series 1 — Foundations (lessons 01–20)
| Lessons | Topic |
|---------|-------|
| 01–04 | Alphabet — uppercase, lowercase, tracing |
| 05–07 | Numbers, days, months |
| 08–10 | Common nouns — body, home, food, animals |
| 11–13 | Greetings and introductions |
| 14–16 | Basic verbs — is, am, are, has, have |
| 17–18 | Colors, shapes, sizes |
| 19–20 | Simple sentences — subject + verb + object |

| Lessons | Topic |
|---------|-------|
| 21–22 | Articles — a, an, the |
| 23–24 | This / that / these / those |
| 25–26 | Question words — what, where, who, when, how, why |
| 27–29 | Present tense — I eat, she runs, they play |
| 30–31 | Past tense — was, were, went, did |
| 32–33 | Future tense — will, going to |
| 34–35 | Prepositions — in, on, at, from, with, by |

| 38–39 | Web vocabulary — click, search, home, login, buy, share |
